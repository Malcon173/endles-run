JURASSIC RUN
1. Visão Geral

Jurassic Run é um jogo de corrida infinita (Endless Runner) ambientado na Era Jurássica. O jogador controla Kan, um jovem explorador preso em uma ilha repleta de dinossauros. Após despertar um enorme T-Rex faminto, Kan precisa correr sem parar para sobreviver.

O objetivo é percorrer a maior distância possível, coletar Dino Moedas, completar missões, desbloquear personagens, melhorar equipamentos e fugir do T-Rex.

4. História

Kan vive em uma misteriosa ilha habitada por dinossauros.

Durante uma expedição, ele desperta a atenção de um enorme T-Rex faminto.

Sem tempo para lutar, Kan deve correr pela selva, atravessar obstáculos, coletar recursos e sobreviver.

Se o T-Rex alcançá-lo, será o fim.

16. Progressão

Com o passar do tempo:

Velocidade aumenta.
Mais obstáculos aparecem.
O T-Rex acelera.
A pontuação cresce mais rapidamente.
17. Sistema de Pontuação

Pontuação =

Tempo de sobrevivência +
Distância percorrida +
Bônus por moedas +
Bônus por sequência sem colisões.

18. Condições de Derrota

O jogador perde quando:

Colide com um obstáculo.
Cai de uma plataforma.
É alcançado pelo T-Rex.

Sobreviver o máximo possível enquanto foge do T-Rex, completar missões, coletar Dino Moedas, melhorar equipamentos, desbloquear novos personagens e estabelecer recordes cada vez maiores em uma aventura jurássica cheia de desafios.
